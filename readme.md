# 📊 Life Insurance Dashboard

A professional **Power BI-style Life Insurance Dashboard** built using **HTML, CSS, JavaScript, Chart.js, and Google Sheets**.

The dashboard automatically reads data from a published Google Sheet (CSV) and updates charts, KPI cards, leaderboards, and tables.

---

# ✨ Features

- 📈 Live Google Sheet Integration
- 🔄 Auto Refresh (30 Seconds)
- 💰 Total Premium
- 📄 Total Policies
- ✅ Issued Policies
- 🏥 Medical Status
- 📍 MAP Status
- 🏢 Company-wise Business
- 👨‍💼 RM Leaderboard
- 📊 Monthly Premium Trend
- 🔍 Search & Filters
- 📋 Live Policy Table
- 📤 Export to CSV
- 📱 Mobile Responsive
- 🌙 Dark Theme

# 🚀 Installation

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/life-insurance-dashboard.git
```

---

## 2. Open Project

Open the folder in **Visual Studio Code**.

---

## 3. Run

Install the **Live Server** extension.

Right-click **index.html**

Select

```
Open with Live Server
```

---

# 📊 Google Sheet Setup

Publish your Google Sheet as CSV.

Example:

```
https://docs.google.com/spreadsheets/d/e/xxxxxxxx/pub?output=csv
```

Replace the URL inside **script.js**.

Example

```javascript
const CSV_URL =
"https://docs.google.com/spreadsheets/d/e/xxxxxxxx/pub?output=csv";
```
